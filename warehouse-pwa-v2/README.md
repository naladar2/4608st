## Шаг 1 — Настроить Google Sheets

### Создать таблицу
1. Откройте [Google Sheets](https://sheets.google.com) и создайте новую таблицу
2. Переименуйте лист в **«Товары»** (или скрипт создаст его сам)

### Подключить скрипт
1. В таблице: **Расширения → Apps Script**
2. Удалите весь существующий код (`function myFunction...`)
3. Скопируйте весь код из файла **`google_apps_script.js`** и вставьте
4. Нажмите 💾 **Сохранить**
5. Нажмите **«Развернуть» → «Новое развертывание»**
   - Тип: **Веб-приложение**
   - Выполнять от имени: **Меня**
   - Кто имеет доступ: **Все** *(важно!)*
6. Нажмите **«Развернуть»**, подтвердите разрешения
7. Скопируйте **URL веб-приложения** (выглядит как `https://script.google.com/macros/s/АБВ.../exec`)

## Шаг 2 — Разместить приложение на хостинге

### Netlify (бесплатно, 1 минута)
1. Зайдите на [netlify.com](https://netlify.com)
2. Перетащите папку `warehouse-pwa-v2` прямо на страницу
3. Получите ссылку вида `https://xxx.netlify.app`

### GitHub Pages (бесплатно)
1. Создайте репозиторий на [github.com](https://github.com)
2. Загрузите все файлы
3. Settings → Pages → Deploy from branch → main
4. Адрес: `https://ВАШ_НИК.github.io/ИМЯ_РЕПОЗИТОРИЯ/`

---

## Шаг 3 — Установить на Android

1. Откройте ссылку в **Chrome** на телефоне
2. Меню **⋮ → «Установить приложение»** или **«Добавить на главный экран»**
3. Иконка появится как обычное приложение

---

## Шаг 4 — Подключить Google Sheets в приложении

1. Откройте приложение → вкладка **«Google Sheets»**
2. Вставьте скопированный URL в поле
3. Нажмите **«Сохранить и проверить соединение»**
4. Нажмите **«Загрузить товары из Sheets»**

## Использование

### Поиск
- Введите штрихкод, артикул или название
- Или нажмите «Сканировать» — направьте камеру на штрихкод
- Нажмите **«Отметить как взятый»** — товар отмечается на карте

### Карта склада
- Отображает все зоны и ячейки
- Синий = найденный товар
- Жёлтый = взят

### Настройки (шестерёнка)
- Добавляйте/удаляйте зоны
- Задавайте цвет каждой зоны
- Меняйте количество ячеек

### Синхронизация
- **↓ Загрузить из Sheets** — получить актуальный список товаров
- **↑ Отправить отметки** — записать «Взято» обратно в таблицу
- Иконка 🔄 в шапке — быстрая синхронизация

Поле «взято» необязательно. Заголовок необязателен.
